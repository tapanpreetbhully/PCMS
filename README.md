# Public Complaint Management System (PCMS)

A web application built with Flask and MySQL to submit and manage public complaints.

## Prerequisites

1. **Python 3.8+** installed on your system.
2. **MySQL Server** installed and running locally.

## Setup Instructions

Ensure that MySQL Server is running locally with the `pcms` database and required tables already configured.

### Application Setup & Launch

1. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python app.py
   ```
3. Access the application at [http://127.0.0.1:5000](http://127.0.0.1:5000).
